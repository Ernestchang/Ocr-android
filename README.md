文字识别，基于Leptonica的深度学习技术，将图片上的文字内容，智能识别成为可编辑的文本。支持印刷体和手写体识别，包括身份证、名片等卡证类，也支持票据、运单等定制化场景的识别，可以有效的代替人工录入信息的场景。提供中英两种语言库,识别准确率高达94%.
